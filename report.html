기획

📌 해결하려 한 문제

업무 중 클립보드는 단 하나의 항목만 기억한다. 여러 셀을 엑셀에서 복사하다 보면 이전에 복사한 텍스트가 사라지고, 다시 원본 파일로 돌아가 재복사하는 과정이 반복된다. 특히 엑셀 데이터를 일부 열·행만 골라 다른 곳에 붙여넣어야 할 때, 기존 클립보드 도구는 아무 도움이 되지 않는다.

😤 Pain Point 1

복사한 내용이 덮어쓰이면 되돌릴 수 없다. 히스토리가 없다.

😤 Pain Point 2

엑셀 범위를 붙여넣으면 전체가 들어간다. 열/행 선택 기능이 없다.

😤 Pain Point 3

클립보드 관리 앱은 대부분 유료거나 너무 무겁다.

목표: 백그라운드에서 조용히 돌아가며, 필요할 때 꺼내 쓸 수 있는 가볍고 빠른 클립보드 매니저. 특히 엑셀 표 데이터 필터링이라는 실무 특화 기능을 핵심으로 삼는다.

구성물 — 주요 기능

🔍

클립보드 자동 감지

0.5초 폴링으로 변경 즉시 감지. 최대 50개 히스토리 보관, 중복 자동 제거.

📊

표 데이터 필터 팝업

엑셀 Ctrl+C 감지 시 자동 팝업. 열/행을 체크박스로 선택 후 붙여넣기.

⚡

자동 붙여넣기

창 숨기기 → 200ms 딜레이 → Ctrl+V 자동 전송. 포커스 충돌 없음.

💾

히스토리 영구 저장

history.json에 저장. 재시작 후에도 이전 히스토리가 복원된다.

🖥

시스템 트레이 상주

X 버튼은 숨기기. 트레이 우클릭으로 열기/종료. 항상 백그라운드 동작.

🔎

실시간 검색 필터

검색창 입력 즉시 히스토리 필터링. 키워드 포함 항목만 표시.

✏️

표 항목 재편집

[편집] 버튼으로 저장된 표 항목을 다시 열어 열/행 선택 상태 복원.

⌨️

전역 단축키

Win+Shift+V로 어디서든 창 열기/닫기 토글. 관리자 권한 필요.

🤖 AI 도구 활용 과정

이 프로젝트는 Claude Sonnet 4.6과의 대화형 개발(Vibe Coding)으로 제작되었다.

  • 초기 설계: 기능 목록을 자연어로 설명하면 전체 구조와 클래스 설계를 제안받아 시작
  • 반복 개선: 표 필터 팝업, 자동 붙여넣기, 트레이 아이콘 등 각 기능을 요청할 때마다 기존 코드를 읽고 수정하는 방식으로 진행
  • 버그 수정: 팝업이 메인 창에 가려지는 문제, 버튼이 기본 크기에서 잘리는 레이아웃 문제, KeyboardInterrupt 전파 문제 등을 에러 메시지 붙여넣기만으로 해결
  • 테스트: 실제 xlsx 파일을 AI에게 분석시켜 헤더 오인식, 푸터 행 포함 등 데이터 품질 이슈를 발견
  • 문서화: requirements.txt, build.bat, README.md, 본 보고서 모두 AI가 생성
코드 한 줄 직접 타이핑하지 않고, 기획·구현·디버깅·문서화 전 과정을 자연어 대화만으로 완성한 100% AI 협업 프로젝트다.

🏗 핵심 아키텍처 결정

  • tk.Listbox → Canvas 커스텀 리스트: 행마다 버튼 배치가 불가능한 Listbox의 한계를 극복하기 위해 Canvas + Frame 조합으로 완전 재구현
  • history를 list[str] → list[dict]: 표 메타데이터(rows, col_states, row_states)와 타임스탬프를 함께 저장하기 위해 구조 변경
  • pack 순서 역전(BOTTOM 우선): tkinter의 공간 배분 특성상 하단 요소(버튼, 미리보기)를 먼저 BOTTOM으로 배치하고 상단 패널이 나머지를 채우도록 설계
  • 자동 붙여넣기 타이밍: 창 withdraw → 200ms 대기 → Ctrl+V → 150ms 후 restore. OS의 포커스 전환 시간을 고려한 값

🔄 핵심 동작 흐름

  1. 프로그램 시작 → history.json 불러오기 → 트레이 아이콘 등록 → 창 숨김(백그라운드 대기)
  2. 사용자가 어디서든 복사(Ctrl+C) → 0.5초 내 감지
  3. 탭 구분 텍스트(표)이면 → 필터 팝업 자동 오픈
  4. 일반 텍스트이면 → 히스토리에 추가 후 저장
  5. Win+Shift+V 또는 트레이 "열기" → 창 표시
  6. 항목 더블클릭 또는 [붙여넣기] → 창 숨기기 → 200ms → Ctrl+V → 창 복원

특장점 & 한계점

✅ 특장점

  • 엑셀 표 필터링: 시중 어떤 무료 클립보드 툴도 제공하지 않는 기능. 원하는 열/행만 골라 붙여넣기 가능
  • 체크 상태 기억: 팝업을 닫아도 열/행 선택 상태가 저장되어 [편집]으로 재오픈 시 복원
  • 자동 붙여넣기: 클립보드 복사 후 포커스 창 전환까지 자동화
  • 영구 히스토리: 재시작 후에도 이전 복사 항목 유지
  • 단일 파일: main.py 하나로 모든 기능 완결. 설치 간편
  • 다크 테마: 눈에 편한 Catppuccin Mocha 컬러 스킴 적용

⚠️ 한계점

  • Windows 전용: pystray, keyboard 라이브러리의 OS 의존성으로 macOS/Linux 미지원
  • 관리자 권한 필요: 전역 단축키 등록에 필수. 권한 없이는 Win+Shift+V 미작동
  • 헤더 오인식: 엑셀에 제목 행(병합셀)이 있으면 실제 컬럼명 대신 제목이 헤더로 인식됨
  • 이미지/파일 미지원: 텍스트 형식만 히스토리에 저장. 이미지 복사는 무시됨
  • 보안: 비밀번호 등 민감 정보도 그대로 history.json에 저장됨
  • 다중 모니터: 팝업 위치가 주 모니터 기준으로만 계산됨
