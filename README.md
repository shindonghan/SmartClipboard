# Smart Clipboard Manager

클립보드 히스토리를 자동 저장하고 표 형식 데이터를 필터링할 수 있는 Windows용 클립보드 관리 도구입니다.

---

## 실행 환경

| 항목 | 요구사항 |
|---|---|
| 운영체제 | Windows 10 / Windows 11 |
| Python | 3.8 이상 (소스 실행 시) |
| 권한 | **관리자 권한 필요** (전역 단축키 등록) |

---

## 설치 방법

### 방법 1 — EXE 파일 실행 (권장)

1. `dist/SmartClipboard.exe` 를 마우스 오른쪽 버튼으로 클릭
2. **"관리자 권한으로 실행"** 선택

### 방법 2 — Python 소스로 실행

```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 관리자 권한 터미널에서 실행
python main.py
```

### 방법 3 — EXE 직접 빌드

```bash
# 1. 의존성 설치 (PyInstaller 포함)
pip install -r requirements.txt

# 2. 빌드 스크립트 실행 (관리자 권한 터미널 권장)
build.bat
```

빌드 완료 후 `dist/SmartClipboard.exe` 가 생성됩니다.

> **수동 빌드 명령어**
> ```bash
> pyinstaller --onefile --windowed --name SmartClipboard ^
>   --hidden-import pystray._win32 ^
>   --hidden-import PIL._imaging ^
>   --collect-all pystray ^
>   --collect-all PIL ^
>   main.py
> ```

## 실행 방법

- **최초 실행**: 관리자 권한으로 실행하면 시스템 트레이에 아이콘이 나타납니다.
- **창 열기**: 트레이 아이콘 우클릭 → "열기" 또는 단축키 `Win + Shift + V`
- **종료**: 트레이 아이콘 우클릭 → "종료"
- 창의 X 버튼은 **종료가 아닌 트레이로 숨기기**입니다.

## 주요 기능

### 클립보드 자동 감지 및 저장
- 0.5초마다 클립보드를 감시하여 변경 시 자동 히스토리 저장
- 최대 50개 항목 유지 (중복 자동 제거, 최신 항목이 상단)
- 프로그램 재시작 후에도 히스토리 유지 (`history.json` 파일에 영구 저장)

### 히스토리 목록
- 각 항목에 복사 시각 표시 (예: `14:23`)
- 더블클릭 또는 [붙여넣기] 버튼으로 해당 내용을 현재 포커스된 창에 자동 입력
- 항목 오른쪽 클릭 → 개별 삭제
- 상단 [전체 삭제] 버튼으로 히스토리 초기화

### 실시간 검색 필터
- 상단 검색창에 입력하면 히스토리를 즉시 필터링
- `✕` 버튼으로 검색어 초기화

### 표(Excel) 데이터 필터 팝업
엑셀에서 셀 범위를 복사(`Ctrl+C`)하면 자동으로 필터 팝업이 열립니다.

- **열 선택**: 필요한 열만 체크박스로 선택
- **행 선택**: 헤더 포함 여부 및 개별 행 선택
- **실시간 미리보기**: 선택 결과를 표 형태로 즉시 확인
- **[편집] 버튼**: 히스토리에서 표 항목을 다시 열어 재편집 가능 (이전 선택 상태 복원)
- **[붙여넣기]**: 선택한 열+행만 현재 창에 자동 붙여넣기
- **[전체 붙여넣기]**: 원본 데이터 전체 붙여넣기

### 시스템 트레이
- 백그라운드에서 상시 동작
- 트레이 아이콘 우클릭으로 창 열기 / 종료

## 단축키

| 단축키 | 동작 |
|---|---|
| `Win + Shift + V` | 창 열기 / 숨기기 토글 |
| 항목 더블클릭 | 클립보드 복사 + 현재 창에 자동 붙여넣기 |
| 항목 우클릭 | 삭제 메뉴 |

---

## 관리자 권한 필요 여부

**관리자 권한이 필요합니다.**

`Win + Shift + V` 전역 단축키 등록에 `keyboard` 라이브러리가 사용되며,
이 라이브러리는 Windows에서 키보드 훅을 등록하기 위해 관리자 권한을 요구합니다.

관리자 권한 없이 실행할 경우:
- 프로그램은 정상 실행되지만 단축키가 작동하지 않습니다.
- 나머지 기능(클립보드 감지, 히스토리, 트레이)은 정상 동작합니다.

## 파일 구조

```
클로드코드/
├── main.py           # 메인 소스 코드
├── requirements.txt  # Python 패키지 목록
├── build.bat         # EXE 빌드 스크립트
├── history.json      # 히스토리 저장 파일 (자동 생성)
├── README.md         # 이 문서
└── dist/
    └── SmartClipboard.exe  # 빌드 후 생성되는 실행 파일
```

## 문제 해결

| 증상 | 해결 방법 |
|---|---|
| 단축키가 안 됨 | 관리자 권한으로 실행 |
| 트레이 아이콘이 안 보임 | 작업 표시줄 트레이 숨김 영역 확인 |
| 표 팝업이 안 뜸 | 이미 팝업이 열려 있는지 확인 (중복 방지) |
| EXE 빌드 실패 | `pip install pyinstaller` 후 재시도 |
